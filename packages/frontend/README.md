# Frontend Package

React + TypeScript + Vite frontend for the SENG 4640 web app.

## Commands

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Environment

- `VITE_API_BASE_URL`
  - Optional backend base URL
  - Defaults to `http://localhost:5000`

## Key Folders

- `src/components/`: page-level and shared UI components
- `src/api/`: backend API clients
- `src/store/`: auth/session state
- `src/constants/`: role constants and shared frontend-only config

Project-wide setup and operations notes live in the repository root
`README.md` and under `doc/`.
